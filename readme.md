# koishi-plugin-zzz

[![npm](https://img.shields.io/npm/v/koishi-plugin-zzz?style=flat-square)](https://www.npmjs.com/package/koishi-plugin-zzz)

Zenless Zone Zero for Koishi | 《绝区零》插件

## 安装

> Koishi 版本需求：^4.15.3

- 通过 `插件市场` 搜索 `zzz` 进行安装。

- 直接安装

``` Shell
yarn add koishi-pligin-zzz
# or use npm:
npm i koishi-pligin-zzz
```
